# token by token

> *If you can't explain something to a first year student, then you haven't really understood .*
> - Richard P. Feynman

We decided to start this project to encourage each other to learn and dive deep into topics in the ML field. By publishing our work, we are held accountable to potential readers to present concepts in a format and language of high quality and one that could be easily understood by other learners. It is called ***token by token*** because we hope to break down and present the information in bits that are digestible, similar to the way we, the writers, learned.

We believe that the best way to learn is to teach, and that we only understand things if we can simplify them and explain them. 

PS: All posts are written entirely by us, humans.
